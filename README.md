# E-Commerce Website
This is a E-Commerce project built with [Next.js](https://nextjs.org/) and [shacdn/ui](https://ui.shadcn.com/).

# Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
**Client:** Next.js, React, TailwindCSS, shacdn/ui
